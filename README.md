## Tweaks

### Amazon Alternatives

- [KOReader](https://koreader.rocks/) - An alternative, highly customisable, E-Book reader
- [Android on Kindles](https://github.com/Ooonana/Guide-to-installing-android-on-kindle) - Replace the OS completely on supported models

### Quality of Life

- [KindleForge](https://github.com/KindleTweaks/KindleForge) - Download Kindle tweaks with ease
- [KindleFetch](https://github.com/justrals/KindleFetch) - Download books from Anna's Archive
- [Disable ADs](https://scriptlets.notmarek.com/scriptlets/disable_ads.sh) - Disable ads on ad-enabled Kindles

### Productivity

- [Textadept](https://github.com/kbarni/textadept-kindle) - Desktop-grade text editor with bluetooth keyboard capabilities
- [KAnki](https://github.com/crizmo/KAnki) - Anki flashcards on Kindles
- [Kreate](https://github.com/Foskya/Kreate) - Unique drawing software for the Kindle

### Games

- [KWordle](https://github.com/crizmo/KWordle) - Kindle Wordle, text-based puzzle
- [IllusionChess](https://github.com/polish-penguin-dev/IllusionChess) - Fast chess
- [Gnome Chess & Minesweeper](https://github.com/crazy-electron/GnomeGames4Kindle) - Slower, chess with more sophisticated AI, classic minesweeper puzzle
- [Gambatte-K2](https://github.com/crazy-electron/gambatte-k2) - Ultra-fast & optimised gameboy emulator with bluetooth audio and multiple rendering modes
- [Crossword](https://github.com/roygbyte/crossword.koplugin) - Kindle crossword, text-based puzzle
- [Tetris](https://kindlemodshelf.me/tetris.html) - Classic game built for kindle, ran in terminal
- [KindleKraft](https://github.com/polish-penguin-dev/KindleKraft) - Soft-float Kindle 1.8 Minecraft server

### Development

- [kTerm](https://github.com/bfabiszewski/kterm) - An e-ink optimised terminal for Kindles
- [Alpine](https://github.com/Entersjkhdfkjdhfksjf/alpine_kindle) - Run Alpine Linux on the Kindle
- [Usbnet-Lite](https://github.com/notmarek/kindle-usbnetlite) - SSH Into your Kindle
- [Illusion](https://github.com/polish-penguin-dev/Penguins-Kindle-Wiki/blob/main/Illusion-Guide.md) - A scriptlet which allows you to create HTML, CSS, JS apps
- [ScreenControl](https://kindlemodshelf.me/screencontrol.html) - Mirrors your kindle's screen online, allows for input

### Miscellaneous

- [KPM](https://github.com/Gingrspacecadet/KPM) - A Kindle Package Manager. Download tweaks directly on-device!
- [SOX Media Player](https://www.mobileread.com/forums/showthread.php?t=368945) - Play bluetooth audio & radio
